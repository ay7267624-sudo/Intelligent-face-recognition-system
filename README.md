# Intelligent Face Detection and Recognition System

## Overview
A real-time face recognition system developed using Python, OpenCV, Flask, and SQLite.

## Features
- Face Detection
- Face Recognition
- Real-time Monitoring
- Database Integration
- Web Dashboard

## Technologies Used
- Python
- OpenCV
- Flask
- SQLite

## Project Outcome
Successfully developed a system capable of detecting and recognizing faces in real time.
